## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

#### Method 1

1. Clone the repo
   ```sh
   git clone https://github.com/JuiciiYT/robloxutil.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```sh
   npm start
   ```

#### Method 2

1. Go to Releases

2. Download the version for your system
